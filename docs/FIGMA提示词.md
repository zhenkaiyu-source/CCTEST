# Figma AI 提示词：瓷器手作创造者平台 UI 设计

---

## 项目背景
为景德镇瓷器手工艺人打造的管理后台，包含作品管理、订单管理、财务管理、数据分析等功能。

---

## 设计风格
- **风格定位**：新中式风格，融合传统陶瓷美学与现代简洁设计
- **主色调**：#8B4513 棕色（陶瓷质感）
- **辅助色**：#D2691E 焦糖色
- **点缀色**：#C9A959 金色
- **背景色**：#F5F0EB 米白色
- **文字色**：#2C1810 深棕色
- **圆角**：8px-20px
- **阴影**：柔和渐变阴影
- **图标风格**：线性图标，2px线宽，圆角端点

---

## 页面设计要求

### 1. 登录页 (/login)
```
布局：居中卡片式
元素：
- Logo（瓷器图案）
- 标题"瓷韵 · 创造者中心"
- 手机号输入框（+86）
- 密码输入框
- 验证码登录切换
- 登录按钮（棕色渐变）
- 微信登录按钮
- 底部"还没有账号？立即注册"
- 背景：景德镇青花瓷纹理或淡雅山水图
```

### 2. 注册页 (/register)
```
布局：居中表单
元素：
- 步骤指示器（1.手机验证 2.填写信息 3.完成）
- 手机号输入
- 短信验证码获取
- 设置密码
- 确认密码
- 昵称输入
- 注册协议勾选
- 注册按钮
```

### 3. 首页仪表盘 (/dashboard)
```
布局：左侧导航 + 主内容区
顶部导航：
- Logo
- 搜索框
- 消息通知铃铛
- 用户头像下拉菜单

概览卡片（4个）：
- 今日收入（金额 + 百分比）
- 本月订单数
- 待发货订单
- 作品总数

快捷入口（4个）：
- 上传作品
- 订单管理
- 财务中心
- 数据分析

图表区域：
- 销售额趋势图（近7天/30天切换）
- 热销作品TOP5

最近订单列表：
- 订单号、作品图、买家、价格、状态、操作按钮
```

### 4. 作品列表页 (/works)
```
顶部工具栏：
- 搜索框
- 分类筛选（下拉：全部/青花/粉彩/颜色釉/雕塑）
- 状态筛选（下拉：全部/已上架/已下架）
- 新增作品按钮

作品卡片网格（响应式）：
- 作品主图
- 作品名称
- 售价
- 销量
- 状态标签
- 编辑/下架/删除操作按钮

分页组件
```

### 5. 上传/编辑作品页 (/works/upload, /works/edit/:id)
```
表单布局：
- 作品主图上传区（拖拽上传，600x600px）
- 作品详情图上传（最多9张，横向排列）
- 视频上传（30秒内，圆形上传按钮）

基本信息表单：
- 作品名称（输入框）
- 作品分类（下拉选择）
- 作品标签（多选：茶具/餐具/花瓶/摆件等）
- 作品描述（富文本编辑器）
- 售价（元）
- 库存数量
- 发货时间（下拉：1-3天/3-7天/7-15天）

操作按钮：
- 保存草稿（灰色）
- 立即发布（棕色渐变）
- 预览按钮
```

### 6. 订单列表页 (/orders)
```
顶部筛选栏：
- 订单状态 tabs（全部/待发货/已发货/已完成/已取消）
- 时间范围选择器
- 搜索订单号

订单卡片列表：
- 订单信息头部（订单号 + 下单时间）
- 作品缩略图 + 名称
- 买家信息
- 订单金额
- 订单状态标签
- 操作按钮（查看详情/发货/联系买家）

分页组件
```

### 7. 订单详情页
```
订单信息区：
- 订单状态流程线（下单 → 付款 → 发货 → 收货 → 完成）
- 物流信息（物流公司 + 运单号 + 查看物流按钮）

作品信息：
- 作品图
- 作品名称 + 规格
- 价格 × 数量

买家信息：
- 收货人
- 联系电话
- 收货地址

操作区：
- 发货按钮（待发货状态）
- 联系买家
- 打印发货单
```

### 8. 财务中心首页 (/finance)
```
概览卡片（3个）：
- 总收入（今日/本周/本月切换）
- 可提现金额
- 冻结金额

收支趋势图：
- 近7天/30天/90天切换

快捷操作：
- 申请提现按钮
- 收支明细入口
- 提现记录入口
```

### 9. 收支明细页
```
列表筛选：
- 类型筛选（全部/收入/提现）
- 时间范围
- 关键词搜索

收支记录列表：
- 日期
- 类型标签（收入/提现）
- 关联订单号
- 金额
- 状态（已完成/处理中/已取消）

分页组件
```

### 10. 提现页
```
账户信息：
- 可提现金额
- 微信账户（已绑定）
- 支付宝账户（已绑定）

提现表单：
- 提现金额输入
- 余额快捷按钮（全部提现）
- 到账方式选择
- 确认提现按钮

提现记录列表：
- 申请时间
- 金额
- 到账方式
- 状态（处理中/已到账/已拒绝）
```

### 11. 数据分析页 (/analytics)
```
数据概览：
- 销售额
- 订单数
- 客单价
- 转化率

图表区域：
- 销售额趋势图（折线图）
- 订单来源分布（饼图）
- 地域分布热力图

排行榜：
- 热销作品TOP10
- 新锐作品TOP5

数据导出按钮
```

### 12. 个人中心页 (/profile)
```
个人信息：
- 头像（圆形，可编辑）
- 昵称
- 手机号
- 认证状态（已认证/未认证）
- 编辑资料按钮

账号安全：
- 修改密码
- 绑定微信
- 绑定支付宝

关于：
- 版本信息
- 帮助中心
- 退出登录按钮
```

## 组件规范

### 按钮
| 类型 | 样式 |
|------|------|
| 主按钮 | 棕色渐变背景，白字，圆角8px |
| 次按钮 | 棕色边框，透明背景，圆角8px |
| 文字按钮 | 无背景无边框，棕色字 |
| 危险按钮 | 红色背景，白字 |
| 禁用按钮 | 灰色背景，50%透明度 |

### 输入框
- 高度：40px
- 圆角：8px
- 边框：1px solid #E5DED3
- 聚焦边框：1px solid #8B4513
- 占位文字：#999999

### 卡片
- 背景：白色
- 圆角：16px
- 阴影：0 4px 20px rgba(0,0,0,0.08)
- 内边距：24px

### 表格
- 表头背景：#F5F0EB
- 行高：56px
- 悬停背景：#FAF8F5
- 边框色：#F0EBE5

### 标签
| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 已上架 | #E8F5E9 | #2E7D32 |
| 已下架 | #F5F5F5 | #666666 |
| 待发货 | #FFF3E0 | #E65100 |
| 已发货 | #E3F2FD | #1565C0 |
| 已完成 | #E8F5E9 | #2E7D32 |
| 已取消 | #FFEBEE | #C62828 |

## 响应式断点

| 断点 | 宽度 | 网格列数 |
|------|------|----------|
| xs | < 576px | 1列 |
| sm | ≥ 576px | 2列 |
| md | ≥ 768px | 3列 |
| lg | ≥ 992px | 4列 |
| xl | ≥ 1200px | 5列 |

## 输出要求

1. **设计稿**：Figma 源文件
2. **组件库**：建立可复用的组件
3. **设计规范**：颜色、字体、间距规范文档
4. **标注**：关键尺寸和间距标注
5. **切图**：@1x、@2x、@3x 切图
6. **动画**：关键交互的微动效设计
