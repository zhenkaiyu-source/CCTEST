## 1. 产品概述

### 1.1 产品定位

**瓷韵** 是一个面向消费者的瓷器手作品电商平台，连接艺术家（创作者）与终端消费者。平台采用C2C模式，让消费者能够直接购买到来自专业艺术家的原创陶瓷作品。

### 1.2 目标用户

| 用户角色 | 目标描述 |
|---------|---------|
| 消费者（C端用户） | 热爱陶瓷艺术、追求个性化家居装饰、购买礼品 |
| 艺术家（B端创作者） | 展示和销售原创陶瓷作品、管理订单、获得收益 |

### 1.3 竞品分析

参考小红书、得物、抖音电商等平台的特点：

| 特点 | 借鉴点 |
|-----|-------|
| 小红书 | 种草笔记、社区氛围、瀑布流展示 |
| 得物 | 商品鉴定、正品保障、潮流属性 |
| 抖音电商 | 直播带货、短视频种草、即时转化 |
| 淘宝/京东 | 标准化购物流程、完善的售后体系 |

### 1.4 设计原则

- **简洁优先**：减少用户决策路径，快速完成购买
- **视觉优先**：陶瓷是视觉艺术，图片质量优先展示
- **信任构建**：艺术家认证、买家评价、售后保障
- **社交属性**：分享、收藏、关注功能

│ • 购物车       │ • 财务管理            │ • 作品审核                │

### 2.2 数据流转

| 数据项 | C端 | B端 | 管理后台 |
|-------|-----|-----|---------|
| 用户 | 读/写个人信息 | 查看买家信息 | 用户管理 |
| 艺术家 | 查看主页/作品 | 个人信息管理 | 认证审核 |
| 作品 | 浏览/购买 | 上传/编辑 | 审核/下架 |
| 订单 | 下单/支付/收货 | 发货/查看 | 监控/退款 |
| 评价 | 发表/查看 | 回复 | 管理 |
| 财务 | 支付记录 | 提现记录 | 结算管理 |

## 7. 页面原型描述

### 7.1 首页原型

**设计风格**：参考小红书瀑布流 + 得物卡片式

**核心区域**：

1. **顶部搜索栏**
   - 固定定位，吸顶效果
   - 搜索框支持语音输入
   - 右侧购物车图标带角标

2. **分类Tab**
   - 横向滚动分类标签
   - 点击切换筛选内容

3. **瀑布流商品**
   - 两列/三列自适应布局
   - 图片高度自适应
   - 底部无限加载

### 7.2 作品详情原型

**核心交互**：

1. **图片画廊**
   - 点击全屏查看
   - 左右滑动切换
   - 支持保存图片

2. **规格选择弹窗**
   - 从底部弹出
   - 支持数量调整
   - 确认后关闭

3. **收藏动画**
   - 点击红心
   - 缩放+粒子效果
   - 状态持久化

### 7.3 下单流程原型

**流程设计**：

```
选择商品 → 加入购物车 → 确认订单 → 选择地址 → 选择支付 → 支付成功
    ↓                                            ↓
  直接购买                                   跳转订单详情
```

**优化点**：

- 减少页面跳转，优先使用弹窗
- 地址选择支持搜索
- 支付成功后自动跳转订单页
