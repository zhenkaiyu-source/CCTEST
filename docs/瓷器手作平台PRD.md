# 瓷器手作平台 - 创造者端 PRD

---

## 一、产品概述

### 1.1 产品名称
**瓷韵 · 创造者中心**

### 1.2 产品定位
为瓷器手工艺人提供作品展示、售卖、财务管理的综合平台。

### 1.3 目标用户
- 景德镇等瓷器产区的独立手工艺人
- 陶瓷艺术家、工作室学徒
- 小型瓷器工作室

### 1.4 核心价值
- 降低手艺人进入电商的门槛
- 提供专业的财务管理工具
- 保护原创作品的版权

---

## 二、用户故事

> **故事1**：作为一位景德镇的瓷器手艺人，我希望能够在线展示和销售我的作品，以便让更多人了解和购买我的手工瓷器。

> **故事2**：作为一位刚入行的学徒，我希望能够看到我的作品销售数据和分析，以便了解市场反馈并改进创作。

> **故事3**：作为一位需要定期回款的手艺人，我希望能够自主申请提现并查看财务状况，以便更好地规划我的创作和生活。

## 六、关键交互

### 6.1 作品上传
1. 点击"上传作品"按钮
2. 填写作品信息（名称、分类、描述、价格）
3. 上传主图（支持拖拽）
4. 上传详情图（最多9张）
5. 上传视频（可选，30秒内）
6. 点击"发布"或"保存草稿"

### 6.2 提现流程
1. 进入"财务中心"
2. 查看账户余额
3. 点击"申请提现"
4. 输入提现金额
5. 选择提现方式（微信/支付宝）
6. 确认提现
7. 等待审核到账（T+3）

### 6.3 发货流程
1. 进入"订单管理"-"待发货"
2. 查看待发货订单
3. 点击"发货"按钮
4. 填写物流公司
5. 填写运单号
6. 确认发货

## 附录

### A. 字段定义
- **可提现金额**：已确认收货且过了售后期的订单金额
- **冻结金额**：待确认收货/售后期的订单金额
- **结算周期**：T+7（订单完成后7天自动解冻）

### B. 术语表
| 术语 | 定义 |
|------|------|
| SKU | 单品规格（如不同尺寸的茶具） |
| SPU | 商品集合（如"青花茶具套装"） |
| T+7 | 订单完成后7天 |

### C. 作品分类
| 分类 | 说明 |
|------|------|
| 青花瓷 | 以青花装饰为主的瓷器 |
| 粉彩瓷 | 以粉彩技法绘制的瓷器 |
| 颜色釉 | 以单一颜色釉装饰的瓷器 |
| 雕塑瓷 | 以雕塑技法制作的瓷器 |
| 日用瓷 | 茶具、餐具等日常用瓷 |
| 艺术瓷 | 纯观赏性艺术品 |
